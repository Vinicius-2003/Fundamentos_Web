Principais Linguagens de programação

Contextualizando

As linguagens de programação são formas de você, usuário, se comunicar com o computador. Assim como os idiomas, servem para estabelecer o contato entre duas pessoas, sendo na Informática esse "contato" feito através de um linguagem de programação.

Nós falamos o idioma português do Brasil, que assim como as linguagens de programação, possui um conjunto de símbolos, as letras do alfabeto. Diferentes maneiras de juntar tais símbolos formam as palavras. A estrutura de organização delas numa sentença é a sintaxe. Tais palavras e frases têm um significado (ou semântica) e o contexto nos permite entendê-lo melhor. Sendo assim, podemos dizer que as linguagens de programação nada mais são do que um idioma com regras de escrita, para garantir a comunicação com o computador!

Linguagens e suas Aplicações

Aqui estão algumas das principais linguagens de programação do mercado e suas aplicações :

  1. Python

    2. A linguagem Python, é a queridinha dos iniciantes na programação, por ser uma linguagem de alto nível, isto é, sua sintaxe é mais proxima de como falamos no dia a dia. Isso implica em uma facilidade em entender seus termos e funções.
    Python pode ser usada nas seguintes áreas :

    • Desenvolvimento web: Python é utilizada para criar sites e aplicações web do lado do servidor, como frameworks Django e Flask.
    • Ciência de dados: Python é muito popular para análise de dados, manipulação de dados, visualização de dados e machine learning, com bibliotecas como NumPy, Pandas entere outras
    • Desenvolvimento de software: Python pode ser utilizada para criar softwares desktop e multiplataforma, como jogos, ferramentas de automação e scripts.
    • Automação de tarefas: Python pode ser utilizada para automatizar tarefas repetitivas, como envio de emails, integração de APIs e criação de scripts.
    • Inteligência artificial: Python é muito utilizada para desenvolvimento de aplicações de inteligência artificial, como chatbots, reconhecimento de imagem e processamento de linguagem natural.

    Python além de ser muito bom para quem está começando a programar, seu ponto forte é o gerenciamento de memória, onde o próprio Python gerencia a memória automaticamente, diferente da maioria das linguagens. Possui uma grande versatilidade, portabilidade e têm uma grande comunidade de desenvolvedores que oferecem supore para a linguagem.

  2. C

    3. A linguagem C, é uma linguagem de nível médio, seu foco maior está no desenvolvimento de jogos e desenvolvimento de sistemas. Porém também é usada nas seguintes áreas :

    • Aplicações embarcadas: C é muito utilizada para programar microcontroladores e outros sistemas embarcados, devido à sua eficiência e portabilidade.
    • Desenvolvimento de jogos: C pode ser utilizada para criar jogos, especialmente jogos 2D e jogos que exigem alto desempenho.
    • Desenvolvimento de sistemas: C é utilizada para criar sistemas operacionais, drivers de dispositivos, firmware e outros softwares de baixo nível.

    Seu ponto forte está na sua eficiência e portabilidade, conferindo a ela ser usada em outros sistemas operacionais como Linux e macOS, e sendo usada para suportar softwares de alto desempenho.

  3. Javascript

    É uma linguagem de alto nível, interpretada e usada principalmente para adicionar interatividade e dinamismo a páginas Web. Ela permite que os desenvolvedores criem scripts que podem :

    • Adicionar interatividade criar animações, jogos, menus suspensos, abas, validações de formulários e outros elementos interativos que melhoram a experiência do usuário.
    • Acessar APIs: interagir com APIs de serviços externos, como Google Maps, Facebook, Twitter, etc., para integrar funcionalidades e dados externos em páginas web. Ajuda a trazer dados que podem nutrir seu site, por exemplo.
    • Controlar multimídia: integrar elementos de áudio, vídeo e imagem, permitindo a reprodução, manipulação e criação de conteúdo multimídia dinâmico.
    • Desenvolvimento de jogos: criar jogos simples ou complexos usando frameworks específicos para jogos em JavaScript, como Phaser.js ou Babylon.js.

    Resumindo, o Javascript, é uma das linguagens mais utilizadas segundo os desenvolvedores. Ela possui uma atuação tanto na área do cliente, isto é o conteúdo que aparece para o usuário ver, como na parte "escondida" de um site.

  4. HTML e CSS

    Embora não sejam lingaugens de programação, e sim lingaugens de marcação e estilização respectivamente, são muito utilizadas tanto em aplicações web como mobile.

    HTML

    • Estrutura o conteúdo: HTML usa tags para definir diferentes elementos da página, como títulos, parágrafos, imagens, listas, tabelas e formulários. Cada tag indica o tipo de conteúdo e sua função.
    • Insere hiperlinks: Uma das funcionalidades mais importantes do HTML é a criação de links (hiperlinks) que permitem ao usuário navegar entre páginas web. Clicando em um link, o usuário é direcionado para outra página relacionada.
    • Define a hierarquia: Tags HTML podem ser aninhadas para criar uma hierarquia de elementos, organizando o conteúdo de forma lógica e facilitando a leitura e interpretação pela navegadores web.
    • Integra conteúdo multimídia: Através de HTML, é possível incorporar imagens, vídeos, áudios e outros elementos multimídia em páginas web, tornando o conteúdo mais rico e engajador.

    É importante lembrar que o HTML não define a aparência visual da página web. Para isso, utilizamos o CSS (Cascading Style Sheets). HTML fornece a estrutura e organização do conteúdo, enquanto o CSS define o estilo visual, como cores, fontes, espaçamentos e layout.

    CSS

    • Formatação visual: CSS define o estilo visual dos elementos de uma página web, como cores de fonte e fundo, tamanho e tipo de fonte, espaçamento entre elementos, margens, posicionamento e efeitos visuais.
    • Separação de conteúdo e apresentação: CSS separa a estrutura do conteúdo (definida em HTML) da sua apresentação visual. Isso permite alterações no estilo da página sem modificar o conteúdo, facilitando a manutenção e atualização do site.
    • Padronização visual: Usando folhas de estilo CSS, é possível criar estilos padronizados que podem ser aplicados a todos os elementos de uma página ou a várias páginas de um site, garantindo consistência visual e facilitando o desenvolvimento e manutenção.
    • Responsividade: CSS permite criar layouts responsivos que se adaptam a diferentes dispositivos, como computadores, tablets e smartphones. Isso é fundamental para garantir uma boa experiência do usuário em qualquer dispositivo.

    Resumindo, o CSS é a linguagem responsável por dar vida e estilo às páginas web criadas com HTML. É como se fosse a "maquiagem" do site, definindo cores, fontes, layout e outros aspectos visuais que tornam a experiência do usuário mais agradável e intuitiva.

Orientação a Objeto

Temos ainda o conceito de orientação a objeto, que é um conceito um pouco mais avançado mas iremos explicar para vocês de uma forma mais intuitiva.

Um objeto, é um "produto" de um molde. O que isso significa ? Quando vamos fazer biscoitos, por exemplo, precisamos de uma forma para colocar o recheio dentro, e assim termos o formato do nosso biscoito.

Na Orientação a objeto, esse molde é a classe. A classe assim como o molde do biscoito, reune as principais características de um objeto. No Natal os biscoitos têm formato de árvore, logo a forma têm as principais "informações" para originar uma árvore : um tronco, o formato e as folhas.O mesmo ocorre na programação, quando criamos uma classe, reunimos as principais características do objeto desejado.

Por exemplo, queremos criar um molde que nos possibilite criar pessoas no nosso código. Logo é necessário capturar as principais informações de uma pessoa.Essa ação é chamada de Abstração. Exemplo : nome, idade, sexo, cpf. Dessa forma, poderei criar quantas pessoas eu quiser usando essas informações que eu define como cruciais no meu código. Faria sentido eu pedir a cor do olho de um pessoas ? Depende do cenário, mas o que no geral, só colocamos na classe as informações mais importantes sobre um objeto.

Tendo entendido esses conceitos, podemos agora entender a aplicação disso.

Aplicação

Com a Orientação a Objeto, podemos dividir nossos códigos em módulos, assim sendo, cada Objeto que você deseje criar um molde, possui um arquivo só para ele, assim facilita a manuntenção do código e facilita o encontro de bugs.
Além disso, diminui o tempo para criar Objetos, pois assim que há uma classe com as principais características de um Objeto, precisamos apenas usar o comando : < nome-da-classe > < nome-do-objeto > = new < nome-da-classe > ();

Assim basta usar esse comando e você já pode manipular um Objeto. Por hora, tenha em mente que a programação hoje, funciona a base de Orientação a Objeto, e que entendendo o conceito, a aplicação é prática, que uma vez dominada, basta aprender a sintaxe da linguagem e você poderá dominar qualquer linguagem de programção que possui a Orientação a Objeto!
